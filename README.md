# pvpbooster_discord_bot
System powiadomień i automatyzacji zarządzania społecznością
Bot integrujący ekosystem PvPBooster z platformą Discord, służący do monitorowania statusu usług i automatyzacji wsparcia.

Kluczowe funkcjonalności:
  * Notification System: Automatyczne raporty o sukcesach/błędach aplikacji przesyłane bezpośrednio do użytkownika.
  * Admin Tools: Komendy do zdalnego zarządzania licencjami i sprawdzania statusu serwerów API.
  * API Integration: Pełna synchronizacja z bazą danych PvPBooster.
